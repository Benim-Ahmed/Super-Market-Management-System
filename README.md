# 🛒 NextGen Supermarket Management System

A **console-based supermarket management system** developed in **C++** as a first semester group project for the *Fundamentals of Computer Programming* course.  
The system automates supermarket operations including **inventory management, billing, cart handling, authentication, and sales reporting** using file handling for persistent storage.

---

## 📌 Project Overview

Traditional supermarket systems often rely on manual record keeping which can be inefficient and error-prone.  
This project introduces an automated console-based solution that simplifies product management, billing processes, and revenue tracking.

The system provides **separate functionalities for Users and Administrators**, ensuring better control and organized operations.

---

## 🎯 Objectives

- Develop a console-based supermarket system using C++
- Automate inventory and billing processes
- Provide secure authentication for users and admins
- Generate invoices and sales reports
- Implement persistent data storage using file handling

---

## 🚀 Features

### 👤 User Functionalities
- Account creation and login
- View available products
- Search and filter products
- Add items to cart
- Generate bills with discounts & GST

### 🛠️ Admin Functionalities
- Add, update, and delete products
- Manage user accounts
- Monitor low stock alerts
- Generate daily, monthly, and total revenue reports

### ⚙️ System Functionalities
- Console-based interface
- Secure authentication system
- Persistent storage using file handling
- Sales and revenue tracking

---

## 🧩 Modules

| Module | Description |

## ▶️ How to Run

Follow these steps to set up and run the project on your system:

### 1. Clone the Repository

First, clone the project from GitHub to your local machine.

```bash
git clone <your-repository-link>
```

### 2. Open the project in your C++ IDE (CodeBlocks / Visual Studio / VS Code)

### 3. Compile and run

```bash
g++ main.cpp -o supermarket
./supermarket
```
